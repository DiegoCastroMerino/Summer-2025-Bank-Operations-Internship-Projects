Within this repository, I've stored VBA code I worked on during my summer 2025 internship at Banco Cuscatlán. This was done as an effort to expedite the customer verification process of recently opened savings accounts. When I started my internship, employees had to manually go through each new customer to make sure they were not missing any documents (ID, forms, tax-related documents, etc.). With my previous background in Excel, I had to create complex macros that would automate this process. 

By the end of my internship, I had cut down the revision process by 95% and the dept. was satisfied with my work
